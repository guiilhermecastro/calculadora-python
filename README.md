# 🧮 Calculadora em Python

## 📌 Sobre o projeto
Calculadora desenvolvida em Python com menu interativo, permitindo realizar operações matemáticas básicas.

## 🚀 Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão
- Validação de divisão por zero
- Menu interativo

## 🧠 Conceitos aplicados
- Estruturas condicionais (if/elif/else)
- Laço de repetição (while)
- Entrada de dados com input
- Lógica de programação

## ▶️ Como executar
1. Abra o terminal
2. Execute:

```bash
python main.py
